Random Walk Simulations

Description: Random walks in 2D and 3D, including variations between cartesian and vector format

Method:
- Generate an array of size n (iterations) x Dimensions (2 for 2D, 3 for 3D)
- Take the cumulative sum over those arrays in their respective columns
- x, y, and sometimes z is extracted from the function and is of size n (iterations)

Results:
- Random walk functions in 2D, 3D, and in vector format
- 5000 iterations, although capable of more

Features:
- Visualization of random walk in 2D and 3D in static and animated graphs
- Utilization of array slicing and NumPy's cumsum function to increase performance for the random walk functions
